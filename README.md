# Veed Interview Task

A frontend app that lists popular repositories on GitHub that were created in the last 7 days.

Demo: https://veed-frontend.vercel.app/

## Installation

```bash
$ yarn
```

## Running tests

```bash
$ yarn test
```

## Running the app

```bash
# watch mode
$ yarn dev
```
